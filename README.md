## Features

- 支持表单校验
- 支持单/多个文件上传
- 支持限制文件格式
- 支持限制文件大小
- 支持限制文件个数
- 支持多种文件格式上传

  - [x] txt
  - [x] zip
  - [x] rar
  - [x] ...
